# 老公别跑（邪恶小女孩黄金矿工）

网页小游戏：你扮演邪恶小女孩，按空格发钩抓目标拿分。

## 当前玩法

- 明星帅哥：体积大、速度中等、分值高
- 我的仓鼠：体积小、速度快、分值中等
- 丑男：体积大、抓取速度极慢、分值极低
- 我老公：速度超级快、分值超级高
- 愤怒章鱼哥：抓到后会喷墨扩散，腐蚀范围内其他目标；抓到会扣分（默认 `-200`）

## 愤怒章鱼哥机制

- 章鱼哥默认扣分值：`200`（结算为 `-200`）
- 喷墨扩散默认半径：`140`
- 配置位置：
  - 独立打开版运行时：`src/main-standalone.js` 中 `GAME_CONFIG.angryOctopusPenalty`、`GAME_CONFIG.octopusInkRadius`
  - 模块版配置：`src/config.js` 中 `DEFAULT_ANGRY_OCTOPUS_PENALTY`、`GAME_CONFIG.octopusInkRadius`

## 账号与登录（MVP）

- 游客可直接开局（桌面与手机端都支持）
- 登录（手机号 + 验证码，Supabase Auth）后可使用排行榜能力
- 登录后检查昵称：
  - 首次登录需设置昵称（1-12 字符）
  - 后续同账号跨设备自动沿用昵称
- HUD 提供“退出登录”，退出后回到游客态（仍可继续游玩）

## 排行与战绩

- 每局结束会提交成绩到 Supabase
- 成绩与 `user_id` 绑定
- 结算页显示“本局全服排名”
- 主页面“战绩查询”可看全服历史 Top15（含游戏时间）

## 本地运行

1. 双击打开 `index.html`
2. 可直接点击开始（游客模式）
3. 若要参与排行榜，再完成手机号登录并设置昵称
4. 桌面端按 `Space` 发钩

## 手机端（iPhone）

- 移动入口：`mobile.html`
- 操作方式：点击屏幕发钩
- 游客可直接开始，登录后可参与排行榜
- 认证服务不可用时不阻塞开局，仅影响登录相关能力

## PWA 安装（iPhone）

前置条件：
- 通过 HTTPS 线上地址访问（例如 Vercel 域名）

安装步骤：
1. 用 iPhone Safari 打开 `https://<你的域名>/mobile.html`
2. 点击“分享”按钮
3. 选择“添加到主屏幕”
4. 从主屏图标打开，进入独立窗口模式（`standalone`）

说明：
- 当前 PWA 主要提供核心静态资源离线可访问能力（页面/脚本/样式等）
- 登录与排行榜等联网能力在离线时会降级

## 本地验证（章鱼哥喷墨）

1. 打开 `index.html`，登录并开始游戏
2. 在水下抓到“愤怒章鱼哥”
3. 观察是否立即出现墨汁扩散动画
4. 检查墨汁范围内其他目标是否被腐蚀消失
5. 等章鱼哥被拉回后，观察分数是否按默认值扣 `200`（分数弹字应显示负数）
6. 执行自动化测试：`npm test`
