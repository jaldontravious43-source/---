# 老公别跑（邪恶小女孩黄金矿工）

网页小游戏：你扮演邪恶小女孩，按空格发钩抓目标拿分。

## 当前玩法

- 明星帅哥：体积大、速度中等、分值高
- 我的仓鼠：体积小、速度快、分值中等
- 丑男：体积大、抓取速度极慢、分值极低
- 我老公：速度超级快、分值超级高
- 愤怒章鱼哥：抓到后会喷墨扩散，腐蚀范围内其他目标；抓到会扣分（默认 `-200`）

## 愤怒章鱼哥机制

- 章鱼哥默认扣分值：`200`（结算为 `-200`）
- 喷墨扩散默认半径：`140`
- 配置位置：
  - 独立打开版运行时：`src/main-standalone.js` 中 `GAME_CONFIG.angryOctopusPenalty`、`GAME_CONFIG.octopusInkRadius`
  - 模块版配置：`src/config.js` 中 `DEFAULT_ANGRY_OCTOPUS_PENALTY`、`GAME_CONFIG.octopusInkRadius`

## 账号与登录（MVP）

- 必须先登录（手机号 + 验证码，Supabase Auth）才能开局
- 登录后检查昵称：
  - 首次登录必须设置昵称（1-12 字符）
  - 后续同账号跨设备自动沿用昵称
- HUD 提供“退出登录”，退出后回到登录态并禁用开局

## 排行与战绩

- 每局结束会提交成绩到 Supabase
- 成绩与 `user_id` 绑定
- 结算页显示“本局全服排名”
- 主页面“战绩查询”可看全服历史 Top15（含游戏时间）

## 本地运行

1. 双击打开 `index.html`
2. 先完成手机号登录
3. 首次登录设置昵称
4. 点击开始，按 `Space` 发钩

## 本地验证（章鱼哥喷墨）

1. 打开 `index.html`，登录并开始游戏
2. 在水下抓到“愤怒章鱼哥”
3. 观察是否立即出现墨汁扩散动画
4. 检查墨汁范围内其他目标是否被腐蚀消失
5. 等章鱼哥被拉回后，观察分数是否按默认值扣 `200`（分数弹字应显示负数）
6. 执行自动化测试：`npm test`
