# 邪恶小女孩黄金矿工（网页版）设计稿

**日期：** 2026-03-30  
**范围：** 先做可玩 MVP（电脑横屏，空格发钩，单关可结算）

## 1. 目标

做一个可在网页直接打开游玩的黄金矿工风格小游戏，玩家扮演“邪恶小女孩”，用钩子抓取不同目标，在倒计时内获得分数。

## 2. 已确认约束

- 平台：电脑横屏（16:9）
- 操作：空格键发钩
- 首版模式：经典闯关（黄金矿工风格）
- 结算：倒计时结束显示分数与达标结果

## 3. 角色与物体设定

- **明星帅哥**：体积大、速度中等、分值高
- **我的仓鼠**：体积小、速度快、分值中等
- **丑男**：体积大、抓取速度极慢、分值极低（非负分）
- **我老公**：速度超级快、分值超级高（难抓但可抓）
- **邪恶小女孩**：主角立绘，位于上方钩子发射点附近

## 4. 核心玩法循环

1. 钩子在上方左右摆动。
2. 玩家按空格，钩子沿当前角度伸出。
3. 命中目标后钩子回收，按目标规则加分。
4. 未命中则触底后回收，不加分。
5. 倒计时结束进入结算层，支持重新开始。

## 5. 参数化设计

每类目标参数化，至少包含：

- `name`
- `sprite`
- `size`（碰撞半径或宽高）
- `moveSpeed`
- `score`
- `pullSpeed`（被抓后回收速度）
- `spawnCount`（初始数量）

建议初始分值区间（后续可调）：

- 明星帅哥：`+300`
- 仓鼠：`+120`
- 丑男：`+20`
- 我老公：`+500`

## 6. UI 结构

- 顶部：分数、目标分、剩余时间
- 中上：邪恶小女孩 + 钩子支点
- 主场景：四类目标随机分布
- 结束层：本局得分、是否达标、重开按钮

## 7. 资源规范

用户上传图片并放入：

- `assets/images/evil-girl.png`
- `assets/images/star-handsome.png`
- `assets/images/hamster.png`
- `assets/images/ugly-man.png`
- `assets/images/husband.png`

要求：

- 建议 PNG（透明背景优先）
- 单图尺寸建议不超过 1024x1024
- 代码内统一缩放到游戏参数尺寸

## 8. 错误与降级

- 图片缺失：回退到彩色占位圆/矩形并给出控制台警告
- 键盘事件失焦：点击画布后恢复监听
- 帧率波动：运动使用 `deltaTime` 计算

## 9. 验收标准（MVP）

- 能在浏览器打开并开始游戏
- 空格可稳定发钩
- 四类物体行为与分值符合定义
- 60 秒结束并正确结算
- 可一键重开
